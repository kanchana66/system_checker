## Features

- CPU Usage (%)
- RAM Usage (%)
- Disk Usage (%)
- Bytes Sent and Received

## Requirements

- Python 3.x
- `psutil` library

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/kanchana66/system_checker.git
   ```
2. Change directory
   ```bash
   cd system_checker
   ```
3. Install dependencies
   ```bash
   pip install psutil
   ```
4. Run the system checker with a simple command
   ```bash
   python system_checker.py
   ```
6. Example Output
   ```java
   System Information:
   CPU Usage (%): 25.0
   RAM Usage (%): 45.2
   Disk Usage (%): 30.5
   Bytes Sent: 10240
   Bytes Received: 20480
   ```
